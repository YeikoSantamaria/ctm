CTM Portal soporta la syntaxis de [Markdown](https://daringfireball.net/projects/markdown/syntax) y tiene unas características adicionales.

## Front Matter

Los documentos de Markdown pueden tener metadatos asociados en la parte superior llamados [Front Matter](https://jekyllrb.com/docs/front-matter/):

```md
---
id: mi-doc
title: Titulo de mi documento
description: Descripción de mi documento
sidebar_label: Mi doc
---

Contenido Markdown
```

## Links Markdown

Los links markdown normales se admiten mediante rutas de url o rutas de archivo relativas.
```md
Vamos ver las [Preguntas frecuentes](/QnA.md).
```

```md
Vamos ver las [Preguntas frecuentes](./QnA.md).
```

Vamos ver las [Preguntas frecuentes](/QnA.md).

## Imagenes Markdown

Se admiten imágenes Markdown regulares y otros formatos.

Agregue una imagen en 'static/img/ctm.png' y utilice esta declaración Markdown:

```md
![Logo de la CTM](/img/ctm.png)

![Logo de la CTM](/img/ctm.webp)
```

![Docusaurus logo](/img/logo_ctm.webp)

## Bloques de codigo

Los bloques de código de Markdown son compatibles con el resaltado de sintaxis.
    ```jsx title="src/components/HelloCtm.js"
    function HelloCtm() {
        return (
            <h1>Hello, CTM!</h1>
        )
    }
    ```

```jsx title="src/components/HelloCtm.js"
function HelloCtm() {
  return <h1>Hello, CTM!</h1>;
}
```

## Admonitions

Docusaurus has a special syntax to create admonitions and callouts:

    :::tip My tip

    Use this awesome feature option

    :::

    :::danger Take care

    This action is dangerous

    :::

:::tip My tip

Use this awesome feature option

:::

:::danger Take care

This action is dangerous

:::

## React components

Thanks to [MDX](https://mdxjs.com/), you can make your doc more interactive and use React components inside Markdown:

```jsx
export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '2px',
      color: 'red',
      padding: '0.2rem',
    }}>
    {children}
  </span>
);

<Highlight color="#25c2a0">Docusaurus green</Highlight> and <Highlight color="#1877F2">Facebook blue</Highlight> are my favorite colors.
```
